### What I learned

Through this project, I learned about CSS Grid, why it exists, how to use it and how is it different from Flexbox so i could determine which one to use depending on the desired layout and that most of the times they could be mixed with eachother just as i did in this project. i also learned different ways or tools to make a layout responsive using grid, media querries...etc but oddly enough the best tool i found interguing to me was the picture element & the image object-fit property, maybe because of their simplicity but yet powerfull effect.

### Continued development
